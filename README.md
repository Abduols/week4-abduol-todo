# Todo List App

## Description

A responsive and professional todo application with the storage persistent ans smooth animation

## Features

- Add new tasks
- Delete task
- Edit task
- Mark completed tasks
- Persistent Storage
- Tasks Load automatically on page refresh
- Smooth animation
- Responsive UI

## Tech Stack

HTML, CSS and Javascript
